# PrintForge

PrintForge — A 3D printable models listing built with Next.js 15 App Router, TypeScript, and Tailwind CSS. 
Browse and search printable 3D models organized by category.
Tech Stack: Next.js 15 · TypeScript · Tailwind CSS

Features:
- Browse models by category
- Search models by name and description
- Loading skeleton UI
- Responsive layout with sticky category sidebar
